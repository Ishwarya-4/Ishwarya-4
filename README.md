### 🚀 Projects I'm proud of

#### [eloqua-ai-email-automation](https://github.com/Ishwarya-4/eloqua-ai-email-automation)
**React · FastAPI · Python · Anthropic Claude API · BeautifulSoup**
An end-to-end marketing automation pipeline that turns JIRA content briefs into production-ready Eloqua email assets. Uses Claude to extract structured content from DOCX briefs, then deterministically builds HTML with BeautifulSoup — AI handles understanding, code handles output. Integrates two real enterprise APIs (JIRA Cloud + Oracle Eloqua) with a 3-step wizard UI.

#### [email-touch-governance](https://github.com/Ishwarya-4/TG-Test)
**React 18 · TypeScript · Supabase · TanStack Query · Vitest** — [Live demo →](https://tg-test1.vercel.app/)
A compliance tool for marketing teams that flags scheduling violations — same-day audience overlaps and email-frequency breaches — across table and calendar views. Built with Zod-validated forms, persistent Supabase storage, bulk JSON import, and a real test suite.

#### [Report_Dashboard](https://github.com/Ishwarya-4/Report_Dashboard)
**React · TypeScript · Vite · shadcn/ui · Tailwind CSS**
An interactive reporting dashboard built with a modern component-driven stack — type-safe end to end, fast dev cycle via Vite.

#### [qkart](https://github.com/Ishwarya-4/qkart)
**React · JavaScript · REST APIs**
A full e-commerce SPA with JWT auth, product search and filters, persistent cart state, and a multi-step checkout — built to mirror real production SPA architecture, not a toy demo.

#### [qtrip](https://github.com/Ishwarya-4/qtrip)
**JavaScript · HTML · CSS · REST APIs**
A dynamic travel booking platform with vanilla-JS DOM rendering, localStorage-based session state, and third-party API integration.
